Documentação de Rotas

  • POST /login: Rota para autenticar um usuário.

    Esta rota permite que os usuários façam login fornecendo suas credenciais.

    Exemplo de solicitação:

            POST /login
        {
          "credentials": {
            "username": "seu-username",
            "password": "sua-senha"
          }
        }
  • POST /bot: Rota para enviar mensagem via WhatsApp.

    Esta rota permite que você envie mensagens via WhatsApp para empresas.

    Exemplo de solicitação:

            POST /bot
        {
          "companies": [
            {
              "celular": "Nome da Empresa 1",
              "razao_social": "Razao Social 1"
            },
            {
              "celular": "Nome da Empresa 2",
              "razao_social": "Razao Social 2"
            }
          ]
        }
  • GET /logout: Rota para fazer logout.

    Esta rota permite que os usuários façam logout e encerrem suas sessões.

    Exemplo de solicitação:

            GET /logout
  • POST /search: Rota para buscar dados.

    Esta rota permite que você busque dados com base no cnae fiscal selecionado.

    Exemplo de solicitação:

            POST /search/cnae
        {
          "cnae": "Cnae Fiscal"
        }
  • GET /: Rota para buscar todos os resultados.

    Esta rota retorna todos os resultados disponíveis.

    Exemplo de solicitação:

            GET /
  • GET /doc: Rota para acessar a documentação de rotas.

    Esta rota redireciona para esta página de documentação.

    Exemplo de solicitação:

            GET /doc
  • POST /search: Rota para buscar dados.

    Esta rota permite que você busque dados com base em diversos critérios de filtro.

    Os critérios de filtro são opcionais e podem ser combinados para refinar a busca.

    Exemplo de solicitação:

            POST /search
      {
        "query":{
          "bairro": "Nome do Bairro",
          "cnae_fiscal": "Cnae Fiscal",
          "motivo_situacao": "Motivo da Situação",
          "natureza": "Natureza",
          "municipio": "Nome do Município",
          "uf": "UF",
          "opcao_mei": "Opção MEI",
          "opcao_simples": "Opção Simples",
          "situacao": "Situação",
          "inicio_atividade": "Data de Início de Atividade",
          "cep": "CEP",
          "user_id": "ID do Usuário"
        }
      }

    Você pode incluir um ou mais critérios de filtro na solicitação. Os resultados serão retornados com base nos critérios fornecidos.
